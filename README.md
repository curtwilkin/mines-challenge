# MINES CHALLENGE — Static Landing

Files in this folder:

- `index.html` — Landing page (headline, subheadline, benefits, CTA)
- `rules.html` — Rules page with exact rules copy
- `styles.css` — Styling for both pages
- `assets/logo.svg` — Simple logo

How to preview locally:

1. Open `MINESCHALLENGE/index.html` in your browser (double-click). OR
2. From a terminal, run a simple local server from the `MINESCHALLENGE` folder:

```bash
python3 -m http.server 8000
# then open http://localhost:8000 in your browser
```
